Aquaculture Keep · rank 3

1. FreshTrack

Post-harvest fish spoilage clock + ice advisor for small wet-market vendors.

Evidence the pain is real

  • SE Asian artisanal fish chains often lose ~15–30% post-harvest (weak cold chain, delayed selling).
  • Globally only ~54% of harvested fish goes to direct human food; ~17% is post-harvest waste.
  • PH framing: Oceana + national cold-storage push aiming for spoilage near 10–15% (implies higher today).
  • Tagum has an active public wet market with fish stalls; ice + open air is the default retail cold chain.

Wedge: one-sided last-mile rules engine — species + hours + ice state → spoilage clock + ice kg + sell / ice more / process / discount. No buyers to recruit. Offline-capable.

Demo loop
Vendor enters species, kg, time since catch, ice on hand → hours-to-risk + ice advice + ₱-at-risk → save → dashboard.
Impact metric

Independent opinion

Keep. Better weekend product than BantayPond for a no-hardware CS team. Pitch answer for BUGSAI: “fisheries value chain ends in Tagum wet market.” Use transparent FAO ice rules — not fake ML.

Circular Agriculture Lock · rank 1

2. SagingWorth

One-sided advisor for Cavendish export-reject bananas — ranks best alt use + ₱ value. Not a marketplace.

Evidence the pain is real

  • Davao del Norte = Banana Capital; clusters include Tagum, Kapalong, Sto. Tomas, Asuncion, Panabo, Carmen.
  • ~5–20% of PH banana crop rejected for not meeting fresh-export specs.
  • Mindanao-scale reject narrative (~1M MT/year) is public industry talk — directionally huge even if ballpark.
  • Local value paths already exist: animal feed (Panabo), chips, flour — but small growers still underprice or dump.
  • Fedco 2025: 10–15% Cavendish production drop + vacant farms in Tagum-area munis.

Wedge: decision ranking, not matching. Reject kg + quality grade → ranked uses (local fresh / chips / feed / compost / flour) with rough ₱/kg.

Demo loop
Optional photo + kg + reject reason → ranked options with ₱ estimate → save decision → dashboard.
Impact metric
₱ recovered from rejects (or kg diverted from dump). Circulab: named waste stream → value.

Independent opinion

Lock as agri default. Strongest Tagum story of the four. Beats BanaLoop for a 48-hour build because it does not depend on two sides. Label ₱ tables as “indicative.” Don’t drift back into marketplace features.

Waste-to-Value Weakest · rank 4

3. DiverTrack

B2G barangay waste-diversion (RA 9003) compliance + improvement tracker. LGU is the first user.

Evidence the pain is real

  • RA 9003: barangays handle bio/recyclable collection; diversion target starts ≥25%; DILG still pushing higher (Aug 2025).
  • Tagum Ordinance 768-s-2016 requires barangays to submit waste diversion + enforcement reports.
  • 2025 peer-reviewed work: solid waste diversion model for LGUs in Davao del Norte.
  • Many barangays still lack functional MRF records (EMB / World Bank SWM assessments).

Independent opinion — weakest of the four

Problem is real. Product is coherent. Buyer is wrong for this event. MSME Development Week judges (DTI, Chamber, Wadhwani) will ask: who is the first user? Answer: barangay secretary / SWM focal — not an MSME. That fights the theme, the Negosyo Center pilot story, and income language. Student “LGU compliance dashboard” is also a hackathon cliché.

Do not lock as waste default. Replace with BasuraBill (below).

Resilience Keep · rank 2 · MSME frame

4. AksyonNgayon

Live anticipatory disaster-action trigger — barangay flood risk + advisory → same-day action list. Not a static BCP.

Evidence the pain is real

  • Tagum sits on the Tagum–Libuganon River Basin (ADB flood-risk projects exist for this basin).
  • Feb 20, 2026: CDRRMO preemptive advisory for 13 named Tagum barangays after Saug / Hijor / Tagum-Liboganon rivers hit critical levels (PNA).
  • Academic flood-preparedness work on Brgy. San Miguel, Tagum.
  • PH institutionalizing anticipatory action (imminent-disaster legislation; WFP / Red Cross protocols).
  • PDRF MSME tools are mostly assessments + guidebooks — not same-day trigger → action lists.

Wedge: forecast/advisory + barangay risk → same-day action list for a business, not a family pamphlet and not a 10-page BCP.

Demo loop

Keep, but sharpen with StallShield framing. Generic barangay flood app loses to CDRRMO. MSME-specific action packs win the room. Seed last real advisory for offline demo at the venue. Don’t claim to replace official warnings.

Original niches (not in prior banks)

A. WiltNext — agri bench, not default

Fusarium / abandoned Cavendish parcel → ranked recovery land-uses + rough time-to-cash. Tagum-area vacant farms are public (Fedco). Longer horizon than SagingWorth → bench #2 only.

B. BasuraBill — waste replacement for DiverTrack

Market stall / carinderia / sari-sari estimates weekly waste by type → app shows tipping cost avoided, diversion options, and ₱ value if sorted (junkshop ranges, compost/BSF as information — not matching).

  • First user = MSME (fixes DiverTrack’s fatal flaw)
  • Circular + Circulab language; metric: kg diverted + ₱ recovered/avoided
  • Competitors (TrashCash/ScrapCycle) are sell-side marketplaces — different job

C. StallShield — resilience framing, not a fifth app

Force AksyonNgayon’s first screen to business type. Deliverables are stall-specific (elevate stock, pre-sell perishables, move ice chest, GCash float, supplier SMS). Same product, correct customer.

Hard disagreements with prior framing

  1. DiverTrack should not sit as a peer-tier default just because RA 9003 is “harder.” Original ≠ win if the buyer is LGU.
  2. BantayPond should not be the mental aqua default for a no-hardware team.
  3. BanaLoop marketplace remains a trap; SagingWorth is the correct agri evolution.
  4. AksyonNgayon without MSME personas will read as another DRRM app — force StallShield framing.
